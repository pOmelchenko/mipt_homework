# Какое время года

Ввести номер месяца и вывести название времени года.

Данные на входе: `Целое число от 1 до 12 - номер месяца.`  
Данные на выходе:
```
Время года на английском: winter, spring, summer, autumn (используйте switch)
```

## Пример №1
Данные на входе: `4`  
Данные на выходе:
```
spring
```

## Пример №2
Данные на входе: `1`  
Данные на выходе:
```
winter
```
